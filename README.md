# V_PL

Концепт видеоплатформы: приём RTSP-видео, привязка к изделию/времени/маршруту, синхронный просмотр видео, карты и телеметрии, поиск по архиву, снимки и метки целей, слой видеоаналитики.

## Запуск концепта

Весь концепт — один файл `index.html` плюс тестовое видео `assets/test-flight.mp4`.

**Важно: открывайте через локальный сервер, а не двойным кликом по файлу.** При открытии напрямую (`file://`) браузер блокирует захват кадров из видео — не будут работать миниатюры и кнопка «Скачать JPG».

```bash
npx serve .
# затем откройте http://localhost:3000
```

Тестовое видео: трейлер Sintel © Blender Foundation, CC BY 3.0 (явно моковый материал).

## Проверки

```bash
npm run verify       # замороженный базовый набор проверок
npm run test:concept # smoke-тест разметки и функций index.html
npm run codex:ship   # полный гейт перед коммитом
```

Правила для агентов и структура скиллов — в `AGENTS.md` (Codex) и `CLAUDE.md` (Claude Code).
