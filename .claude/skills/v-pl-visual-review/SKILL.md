# V_PL Visual Review

Do not judge visual quality from source alone. Require either:

- a screenshot path, or
- a local URL that can be opened and captured.

Assess whether the view reads as a serious operator tool for video, map,
telemetry, archive, and analytics workflows. Focus on density, hierarchy,
legibility, synchronized state, and absence of decorative clutter.
