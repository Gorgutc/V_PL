# V_PL Code

## Quick Workflow

1. Read `AGENTS.md`, then `docs/agent/architecture.md` and
   `docs/agent/verification.md`.
2. Identify whether the work changes a local concept, verification logic, or
   agent infrastructure.
3. Preserve product/time/video/map/telemetry contracts.
4. Run `npm run verify` after shipped-code or verification edits.

Use mock data for local concepts unless approved real data is provided.
