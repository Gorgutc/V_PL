# V_PL Data Contracts

Every durable record should be traceable to product and time. Prefer explicit
IDs and timestamps over implicit UI state.

Minimum conceptual entities:

- product
- video source
- video segment
- frame snapshot
- label or object marker
- route point
- telemetry sample
- analytics artifact
